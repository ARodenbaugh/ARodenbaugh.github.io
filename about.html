Acronym Database: About

 
 
            This site is a reference tool to assist people in becoming familiar with commonly used acronyms at
        the Laboratory for Atmospheric and Space Physics. Originally created as a final training project for a
        student employee at LASP, this site will hopefully continue to be used by new hires as a way to look up
        acronyms as they are introduced to them.

            Along with this website, the project consists of an underlying database that stores all of the acronym
        information. There are also two separate GUI's that allow the adding of new acronyms to the database and
        the editing/deleting of existing acronyms from the database. With any edit to the database using these tools,
        the website is automatically updated through the live server.
